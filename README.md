# 功能
## 1.实时显示
显示对齐后的 RGB 图像和深度伪彩图像。
## 2.数据保存
按 s 保存 RGB-D 数据到 .mat 文件  
按 a 保存点云到 .ply 文件  
退出程序
按 q 或 ESC 退出
# Python 环境
Python 版本：3.7 - 3.10 
# 依赖库
`pip install -r requirements.txt`
# 安装 Intel RealSense SDK
从 [ Intel RealSense GitHub ](https://github.com/IntelRealSense/librealsense/releases)下载最新版本的 SDK。  
需要下载完整版
